# Keyboard Shortcuts

Complete reference for all keyboard shortcuts in Tappi Browser.

> **macOS notation:** `⌘` = Command, `⇧` = Shift, `⌃` = Control, `⌥` = Option.  
> `CmdOrCtrl` means `⌘` on macOS and `Ctrl` on Windows/Linux.

---

## Navigation

| Shortcut | Action |
|----------|--------|
| `⌘L` | Focus the address bar |
| `⌘R` | Reload the current page |
| `⌘[` | Go back |
| `⌘]` | Go forward |

---

## Tabs

| Shortcut | Action |
|----------|--------|
| `⌘T` | Open a new tab |
| `⌘W` | Close the active tab |
| `⌘⇧T` | Reopen the last closed tab |
| `⌘1` | Switch to tab 1 |
| `⌘2` | Switch to tab 2 |
| `⌘3` | Switch to tab 3 |
| `⌘4` | Switch to tab 4 |
| `⌘5` | Switch to tab 5 |
| `⌘6` | Switch to tab 6 |
| `⌘7` | Switch to tab 7 |
| `⌘8` | Switch to tab 8 |
| `⌘9` | Switch to the last tab |

---

## Page Tools

| Shortcut | Action |
|----------|--------|
| `⌘F` | Find on page |
| `⌘P` | Print current page |

## Panels & Views

| Shortcut | Action |
|----------|--------|
| `⌘J` | Toggle the agent panel (sidebar) |
| `⌘Y` | Open the History panel |
| `⌘⇧B` | Open the Bookmarks panel |
| `⌘⇧D` | Open the Downloads panel |
| `⌃⌘F` | Toggle fullscreen (window-level) |

| Shortcut | Action |
|----------|--------|

---

## In-Page Shortcuts (when a web page is active)

These are handled by Electron's WebContents standard behaviour:

| Shortcut | Action |
|----------|--------|
| `⌘+` | Zoom in |
| `⌘-` | Zoom out |
| `⌘0` | Reset zoom |
| `Space` | Scroll down |
| `⇧Space` | Scroll up |

---

## Developer Tools

| Shortcut | Action |
|----------|--------|
| `⌘⌥I` | Toggle Chrome DevTools for the active page (via View → Toggle DevTools) |

## mpv Media Overlay (when mpv window is focused)

These are mpv's built-in shortcuts — active only when the mpv overlay has keyboard focus:

| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `←` | Seek back 5 s |
| `→` | Seek forward 5 s |
| `↑` | Volume up |
| `↓` | Volume down |
| `m` | Mute / unmute |
| `f` | Toggle fullscreen |
| `q` | Quit mpv overlay |
| `[` | Decrease playback speed |
| `]` | Increase playback speed |
| `9` | Volume down (alternate) |
| `0` | Volume up (alternate) |
